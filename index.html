Changes

  1. Cario talks about two different types of data in chapter 7 and chapter 8 of his book. The data he talks about in chapter
    7 is data that is gathered from experiments and surveys. This data can be manipulated to show trends such as the mean and
    median. In chapter 8, however, the data he describes is collected and shows trends over time.
  2. Kinds of Visualizations
    • Seasonal Subseries Plot- a chart used to display the month-by-month variation of a given variable. This type
      of visualization is useful because you can compare the monthly variations to the monthly and yearly averages. However,
      this type of plot can be difficult to read if you have not seen this type of plot before.
    • Logarithmic Scale- a chart that can show the percentage increase or decrease per day of a given variable. These are
      useful to show the rate of change of a variable each day relative to days before it. This type of visualization, however,
      does not allow us to see the overall change for the given time frame of the variable.
    • Connected Scatterplot- a scatterplot that shows how each point of the data set is connected by a line. These plots are
      help show how closely the data is reated to one another. There becomes a problem with this plot when there are turns and
      swirls that can throw off the data.
